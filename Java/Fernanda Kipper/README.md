## Aprendizados:

- Todo arquivo java é obrigado a declarar uma classe.
- O nome da classe deve ser o mesmo nome do arquivo, e ela deve ser public

- Declarar sempre o tipo da variavel. Ex:
tipo (sendo ela string, int..) nomeVariavel = valor

- Pode usar a variavel "var", ela infere automaticamento o tipo da variavel, ex:
var nomeVariavel = "Gabriel"

**Tipos**
- número inteiros:

byte => 8 bits => -128 a 127
short => 16 bits -32.768 a 32.767
int => 32 bits -2.147.483.648 a 2.147.483.647
long => 64 bit -9.223.372.036.854.775.808 a 9.223.372.036.854.775.807

- números decimais:

float => 32 bits -> precisão simples, ex 1,52
douvle => 64 bits -> precisão dupla

String => representa palavras e frases

char => representa um únicocaractere => a, v, g, d
        
boolean => true ou false
