# Estudos sobre Git e Github
Aqui irei colocar meu conhecimentos adquiridos e guias de como fazer e utilizar essa incrivel ferramenta

| comando | função/resumo | Exemplo |
|------|-------|------|
| git add . | adcionar arquivo | Eiii git, tem arquivo novo aqui hien |
| git commit -m"" | Comentar alterações | Salva essa versão com essa etiqueta, ex: "aula 01 - aprendendo musica" "alteração no código" |
| git push | Enviar para a núvem | Enviar e salvar no github |
|cd | Seria "Change Directory", para mudar a pasta dentro do terminal | estou na pasta /menu e quero entrar na subpasta /sobre, ai digiratia "cd sobre" ou "cd C:/Usuarios/SeuNome/menu/sobre"
|cd ..| Volta para a pasta anterior | no exemplo acima, após utilizar esse comando na pasta /sobre eu voltaria para a pasta /menu
